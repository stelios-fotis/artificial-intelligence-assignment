# artificial-intelligence-assignment

This is an assignment for Artificial Intelligence CN6005.
It trains a simple neural network to predict heart disease from the [UCI Heart Disease Cleveland dataset](https://archive.ics.uci.edu/dataset/45/heart+disease).

## Instructions

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python main.py
```
